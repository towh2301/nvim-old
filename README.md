# Neovim Config

- This is requirement
  - Require NerdFont
  - Nodejs
  - Npm
  - Pip

### For window OS

- If errors with fzf-native just install with choco or winget

```
choco install mingw -- optional
choco install make
```

- In the directory `\AppData\Local\nvim-data\site\pack\packer\start\telescope-fzf-native.nvim` run

```
make
```
